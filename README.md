# OSCP and CTF Tools Collection

This repository contains a collection of binaries and tools that I have utilized during my OSCP (Offensive Security Certified Professional) certification and various Capture The Flag (CTF) challenges. These tools have been invaluable for various tasks such as penetration testing, privilege escalation, exploitation, and post-exploitation.

## Tools List

- **CrackMapExec**
  - A Swiss army knife for pentesting Windows/Active Directory environments.

- **DecryptAutoLogon.exe**
  - Tool used to decrypt saved credentials for auto logon.

- **GodPotato-NET4.exe**
  - A privilege escalation tool exploiting COM services.

- **Invoke-Mimikatz.ps1**
  - A PowerShell script to run the famous Mimikatz tool for credential dumping.

- **JuicyPotato.exe**
  - A privilege escalation tool for Windows exploiting the token duplication vulnerability.

- **JuicyPotatoNG.exe**
  - An updated version of JuicyPotato for Windows privilege escalation.

- **KiteService.exe**
  - A tool used for privilege escalation.

- **PowerShellGenerators**
  - A collection of PowerShell scripts for various offensive tasks.

- **PowerUp.ps1**
  - A PowerShell script for privilege escalation on Windows machines.

- **PrintSpoofer64.exe**
  - A tool for privilege escalation using the Print Spooler service.

- **PrivescCheck.ps1**
  - A PowerShell script to enumerate possible paths to escalate privileges on a Windows machine.

- **Rubeus.exe**
  - A tool for Kerberos abuse, including ticket requests and renewals.

- **RunasCs.exe**
  - A tool for executing commands with different user privileges.

- **RunasCs_net2.exe**
  - Another variant of RunasCs for specific .NET environments.

- **Seatbelt.exe**
  - A C# project that performs a number of security-related checks on a Windows system.

- **SharpHound.ps1**
  - A PowerShell script for gathering Active Directory information for BloodHound.

- **SweetPotato.exe**
  - Another privilege escalation tool similar to JuicyPotato.

- **checkDisabledFunc.php**
  - A PHP script to check for disabled functions in PHP environments.

- **chisel**
  - A fast TCP tunnel, transported over HTTP, secured via SSH.

- **directoryLists**
  - A collection of directory listings used for web application fuzzing.

- **dirsearch**
  - A simple command line tool designed to brute force directories and files in web servers.

- **evil-winrm**
  - The ultimate WinRM shell for hacking/pentesting.

- **ferox-http_192_168_201_48-1703180422.state**
  - A state file for Feroxbuster, a tool for directory discovery on web servers.

- **gdb_commands**
  - A collection of GDB commands and scripts for debugging.

- **git-dumper**
  - A tool to dump a git repository from a website.

- **impacket**
  - A collection of Python classes for working with network protocols.

- **joomscan**
  - A vulnerability scanner for Joomla CMS.

- **kerbrute_linux_amd64**
  - A tool to quickly bruteforce and enumerate valid Active Directory accounts.

- **kr**
  - A tool for remote command execution via SSH.

- **ligolo-agent**
  - Ligolo is a simple, lightweight, and fast reverse-tunneling tool optimized for penetration testers.

- **linpeas.sh**
  - A script that searches for possible paths to escalate privileges on Linux/Unix systems.

- **mimikatz.exe**
  - A tool for dumping credentials from Windows systems.

- **mimikatz64.exe**
  - The 64-bit version of Mimikatz.

- **mimikatz_trunk**
  - The development branch of Mimikatz with the latest features.

- **nc**
  - Netcat, a versatile networking tool.

- **nc.exe**
  - The Windows version of Netcat.

- **nc64.exe**
  - The 64-bit version of Netcat for Windows.

- **nmap.exe**
  - A network scanning tool for discovering hosts and services on a computer network.

- **nmap**
  - The Linux version of Nmap.

- **ntlm_theft**
  - Tools for capturing NTLM hashes.

- **passwordList**
  - A collection of commonly used passwords for brute-force attacks.

- **pentestmonkey**
  - A collection of pentesting scripts.

- **plink.exe**
  - PuTTY Link: a command-line connection tool similar to SSH.

- **powercat.ps1**
  - A PowerShell TCP/IP swiss army knife, a tool for network interactions.

- **powerview.ps1**
  - A PowerShell tool to gain network situational awareness on Windows domains.

- **pspy64**
  - A tool to monitor Linux processes without root permissions.

- **pypykatz**
  - A Python implementation of Mimikatz to extract credentials from memory dumps.

- **reports**
  - A directory to store penetration testing and CTF reports.

- **scripts**
  - A collection of scripts used during pentesting and CTF challenges.

- **snmpbrute.py**
  - A script to brute force SNMP community strings.

- **socat**
  - A relay for bidirectional data transfer between two independent data channels.

- **unix-privesc-check**
  - A script to check for common misconfigurations and vulnerabilities for Unix privilege escalation.

- **winPEAS.bat**
  - A batch script to check for common misconfigurations and vulnerabilities for Windows privilege escalation.

- **winPEASx64.exe**
  - The 64-bit version of winPEAS for Windows privilege escalation.
